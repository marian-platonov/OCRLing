OCRLing Release Notes

v1.0.1

What's New

  • Menu tray
    Right-click on the OCRLing icon to open the accessible menu options, including Select area, Select image, QR & Barcode Reader, Desktop Color Picker, Settings, About, Help, and Exit.
  • OCR Loading Animation
    An animated loading indicator is now displayed over the OCR window while text extraction is in progress.
    • The animation appears automatically when Process OCR is clicked and disappears as soon as results are ready.
    • OCR processing runs on a background thread so the UI remains responsive during extraction.
    • A status label below the animation confirms processing is underway.
  • Scrollable Result Windows
    All result windows (Select Area OCR and Select Image OCR) now include a vertical scrollbar.
    • Windows can be resized to any size — all content remains accessible by scrolling.
    • Frames stretch horizontally to fill the window width automatically on initial load.
    • The mouse wheel scrolls the window content when the cursor is inside the window; when hovering over the image, the scroll wheel zooms the image instead.
  • Application Logging
    This tool enables event logs to be displayed in message boxes and stored in the ./logs/application.log file.
    • Log format: 
      {time} | {process} | {level} | {file} | {module}:{function}:{line} | {message}
    • Logs are rotated daily.
    • Logs are retained for one month.
  • Select area
    • After selecting Select area, a draggable rectangle appears on the screen, allowing you to capture a specific section as a temporary image for OCR analysis.
    These features are included:
    • View loaded image
    • Use the zoom-in and zoom-out functions to adjust the image's magnification, allowing you to examine details more closely or return to a wider view.
    • Hold and drag the image to move around and explore its details.
    • Copy Image to Clipboard – Allows you to copy the image for use in other applications.
    • Save Image As... – Allows saving the image in a different format and location of your choice.
    • OCR Text Extraction – Extracts text from images using either Tesseract OCR or Mistral AI OCR, depending on the selected engine.
    • Text Language Detection – Identifies the language of the provided text using advanced linguistic analysis techniques.
    • Error Handling – Manages unexpected issues during processing, ensuring stability and providing relevant error messages for troubleshooting.
    • The Extracted Text editable box provides several menu options for managing text:
      • Select All – Highlights all extracted text for quick selection.
      • Copy – Copies the selected text to the clipboard for use in other applications.
      • Cut – Removes the selected text and places it on the clipboard for pasting elsewhere.
      • Paste – Inserts the text from the clipboard into the editable box.
      • Undo – Reverts the last action, restoring the previous text state.
      • Save – Saves the extracted text to a file for future reference.
    • The Translated Text editable box provides several menu options for managing text:
      • Select All – Highlights all extracted text for quick selection.
      • Copy – Copies the selected text to the clipboard for use in other applications.
      • Save – Saves the extracted text to a file for future reference.
    • The Source/Target Language combo box displays a list of all languages (245) supported by the googletrans library. It allows manual selection or language search with auto-complete functionality.
    • Convert the text from the Extracted Text box into the Translated Text box using the googletrans library.
    • Clear button removes text from both the Extracted Text and Translated Text boxes, resets the Source Language and Target Language to English, and updates the Detected label to Detected: Not analyzed.

  • QR & Barcode Reader
    • Selecting QR & Barcode Reader opens the browser-based reader in your default system browser.
    These features are included:
    • Browser-based tool — no image data is ever uploaded to a server.
    • Two input modes: Image mode (load from file, drag-and-drop, or clipboard) and Camera mode (live webcam scanning).
    • Supports 2D codes (QR Code, Aztec, Data Matrix, MaxiCode, PDF-417) and 1D barcodes (Code 39, Code 93, Code 128, EAN-8, EAN-13, UPC-A, UPC-E, Codabar, ITF, RSS-14, RSS Expanded, UPC/EAN Extension).
    • Image enhancement controls: Brightness, Contrast, Binarize, Grayscale, Invert, and Sharpen — applied in real time before scanning.
    • Decoded results shown in the Decoded Result panel with format badge, character count, and full decoded text.
    • Copy decoded text to clipboard or open decoded URLs directly in a new tab.
    • Camera mode scans continuously at 15 fps with a green highlight box on detected codes.
    • Dark/light theme toggle and built-in user guide overlay.
  • Desktop Color Picker
    • Once the Desktop Color Picker is selected, the below page will open:


    These features are included:
    • Pixel Color Detection: Click anywhere on your screen to capture the exact color of that pixel
    • Color Wheel Interface: Interactive color wheel for manual color selection
    • Multi-Format Display: Shows color values in Hex, RGB, and HSV formats simultaneously
    • Real-time Updates: All color formats update automatically when you select a new color
    • Clipboard Integration: One-click copying of hex values for immediate use
  • Settings
    • Settings allows to set or control:
    • The default OCR engine can be selected between Tesseract OCR and Mistral OCR.
    • The Mistral AI API Key text field allows to store it for later usage. It will take effect only when the Mistral OCR is used as the default OCR engine.
    • The Default tesseract.exe path is a non-editable text field that stores the full local path to tesseract.exe. It will take effect only when the Tesseract OCR is used as the default OCR engine.
    • Reset button restores the application to its original settings, setting Tesseract OCR as the default engine, removing the Mistral API Key, and clearing the local tesseract.exe path.
    • Restart button allows application restart
  • About
    • About page will have some short information about this tool.
  • Help
    • Help option will open the ./documentation/help.html webpage in the default system browser, allowing to review the app documentation.
  • Exit
    • Exit option will close the application along with any open pages.
